# WooCommerce Payment Privacy Shield

Advanced privacy for high-risk WooCommerce stores.

Hides your site URL/referrer and shows **only Order Number + Amount + Real Customer Name** on payment gateway pages.

## Features
- Real customer name from billing
- Secure proxy (`/pay-proxy/`) with nonce + whitelist
- Server-side `Referrer-Policy`
- Webhook verification with retry
- Supports 20+ gateways

## Installation
Upload to `/wp-content/mu-plugins/`
